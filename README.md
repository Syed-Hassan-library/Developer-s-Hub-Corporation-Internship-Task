## Task 1 — Iris Dataset Exploration

📂 **Folder:** [Task 1 Iris\_Dataset\_Exploration](./Task%201%20Iris_Dataset_Exploration/)

| Field | Details |
|-------|---------|
| **Dataset** | Iris Dataset (UCI ML Repository) |
| **Type** | Exploratory Data Analysis (EDA) |
| **Techniques** | Data Visualization, Statistical Analysis, Feature Analysis |
| **Libraries** | Pandas, Matplotlib, Seaborn, NumPy |

**Description:**  
Performed complete EDA on the classic Iris dataset. Analyzed feature distributions, correlations, and visualized class separability using pair plots, heatmaps, and box plots.

## Task 3 — Heart Disease Prediction

|-------|---------|
| **Dataset** | Cleveland Heart Disease Dataset |

| **Techniques** | Logistic Regression, Random Forest, SVM |
| **Libraries** | Scikit-learn, Pandas, Seaborn, Matplotlib |
